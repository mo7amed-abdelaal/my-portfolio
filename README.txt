# Mohamed Portfolio (Ready-to-Run)

## Quick Start (Windows)
1. Open a terminal **inside this folder**.
2. Run: `npm install`
3. Run: `npm run dev`
4. Open the URL printed by Vite (usually http://localhost:5173/).

This project uses:
- React + Vite
- Tailwind CSS (v3)
- framer-motion
- lucide-react

If you see errors:
- Delete `node_modules` and `package-lock.json`, then run `npm install` again.
- Make sure Node.js >= 18 (Node 20 recommended).
